# La Rocca v1.0 Frontend

Complete bilingual static frontend for GitHub Pages.

## Included
- Full English/Spanish translation
- Automatic Open/Closed status
- Live-style status card
- Countdown to closing/opening
- Today's hours
- Cleaning override for testing
- Announcement banner
- Gallery modal
- WhatsApp and Google Maps links
- Mobile responsive layout

## Configure
Edit `js/config.js` for WhatsApp, Maps, announcements, and temporary manual status override.
Edit `js/schedule.js` for the real gym hours.

The default hours are examples only.
